# Vectorization Benchmark (NumPy vs Python Loops)
A simple study on how NumPy vectorization improves computation speed over Python loops


## 📌 Overview

This project compares the performance of standard Python loops with NumPy vectorized operations. It demonstrates how vectorization improves execution speed in numerical computations.

## ⚙️ Tools Used

* Python
* NumPy
* time (for benchmarking)

## 🧪 What I Did

* Implemented a basic numerical operation using Python loops
* Re-implemented the same operation using NumPy vectorization
* Measured execution time for both approaches
* Compared performance differences

## 📊 Key Insight

Vectorized operations in NumPy are significantly faster than Python loops because they use optimized low-level implementations.

## 🚀 How to Run

```bash
python vectorization_demo.ipynb
```
